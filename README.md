<br>

# Build and run instructions

<br>

## Project init (after clone or fork)

```bash
make init
```

<br>

## Project run in development mode

```bash
make dev-run
```

<br>

## Project to docker container build

```bash
make docker-build
```

<br>

## Docker container run

```bash
make docker-run
```
